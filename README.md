# Fly Agile Brand Assets

Brand design system, guidelines, and source files for Fly Agile (by Giom Consulting).

## Contents

- `design-system.md` — Brand DNA, colours, typography, logo guidelines, taglines, art direction
- `voice-and-tone.md` — Voice, archetypes, style rules, LinkedIn vs long-form
- `CHANGELOG.md` — History of design system evolutions
- `Brand Guide/` — Adobe Illustrator source file for the full brand guide (2020)
- `Logo/` — All logo variants (AI, EPS, PSD) across formats, backgrounds, and use cases
- `notes/` — Brand study notes and research PDFs

## Logo taglines

- "Fly Agile with Giom" — standard
- "Fly the Agile Life" / "Fly the Agile Life with Giom" — lifestyle variants
- "Professional Services" — services variant
